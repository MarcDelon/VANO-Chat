# 🚀 VANO-CHAT - Le Réseau Social Moderne

Bienvenue sur **VANO-CHAT**, une plateforme sociale complète développée dans le cadre du module **API REST**. Ce projet combine une architecture backend robuste (Node.js/Express) et une interface frontend dynamique et élégante (Next.js).

## 🛡️ Caractéristiques Techniques

- **API RESTful** : Plus de 22 endpoints gérant les utilisateurs, les posts, les likes, les stories et la messagerie.
- **Sécurité** : Authentification par **JSON Web Tokens (JWT)** et hachage des mots de passe avec **Bcrypt**.
- **Base de Données** : Infrastructure Cloud performante avec **Supabase (PostgreSQL)**.
- **Interface UI/UX** : Design moderne avec **Tailwind CSS v4** (Glassmorphism & Dark Mode).
- **Temps Réel** : Suivi des notifications et discussions privées.

## 🚀 Installation et Lancement

### 📦 Pré-requis
- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/)

### 🛠️ Backend
1. Allez dans le dossier backend :
   ```bash
   cd Backend
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Configurez votre fichier `.env` (voir `.env.example`).
4. Lancez le serveur :
   ```bash
   npm run dev
   ```

### 💻 Frontend
1. Allez dans le dossier frontend :
   ```bash
   cd Frontend
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez l'application :
   ```bash
   npm run dev
   ```
4. Accédez à l'app sur `http://localhost:3000`.
